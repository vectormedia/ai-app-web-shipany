# ShipAny Template Two - 功能总结

## 项目概述

ShipAny Template Two 是一个现代化的全栈 SaaS 应用模板，专为 AI 驱动的内容生成平台设计，集成了完整的支付处理系统。该模板支持多种部署目标（Vercel、Cloudflare、Docker）和多种支付提供商（Stripe、PayPal、Creem）。

## 核心功能模块

### 1. AI 内容生成系统
- **图像生成**：使用多种 AI 提供商的图像创建功能
- **音频生成**：AI 音频合成能力
- **音乐生成**：AI 音乐创作工具
- **视频生成**：AI 视频创建功能
- **聊天机器人**：对话式 AI 接口，支持聊天历史记录

### 2. 用户认证与授权
- **多种登录方式**：邮箱密码登录、OAuth（Google、GitHub）
- **角色权限控制**：基于 RBAC 的细粒度权限管理
- **会话管理**：安全的用户会话处理
- **API 密钥管理**：用户自生成的 API 密钥用于外部访问

### 3. 订阅与支付系统
- **多支付提供商**：Stripe、PayPal、Creem 集成
- **订阅管理**：循环计费和订阅状态管理
- **积分系统**：基于代币的使用跟踪，支持积分过期
- **账单管理**：完整的账单历史和发票系统

### 4. 管理后台
- **用户管理**：用户信息管理和权限分配
- **支付监控**：交易监控和财务数据分析
- **内容管理**：CMS 风格的内容管理系统
- **系统配置**：动态配置 OAuth、支付和功能开关

### 5. 内容管理系统
- **文章管理**：支持 MDX 的文章创建和编辑
- **分类系统**：层级化的分类和标签管理
- **SEO 优化**：自动生成 SEO 元数据
- **多语言支持**：英文和中文本地化

### 6. 聊天系统
- **实时对话**：AI 聊天功能
- **历史记录**：聊天记录持久化存储
- **会话管理**：多个聊天会话管理

## 技术特性

### 1. 国际化支持
- **多语言路由**：`/en/` 和 `/zh/` 语言前缀
- **动态翻译**：基于 next-intl 的翻译键管理
- **自动语言检测**：根据用户偏好自动选择语言

### 2. 主题系统
- **动态主题**：基于 CSS 变量的主题切换
- **组件主题化**：所有组件支持主题定制
- **Tailwind 集成**：与 Tailwind CSS 深度集成

### 3. 数据库设计
- **多数据库支持**：PostgreSQL、SQLite、MySQL
- **类型安全**：Drizzle ORM 提供完整的类型安全
- **迁移管理**：自动化数据库迁移系统
- **性能优化**：完整的索引策略和查询优化

### 4. API 设计
- **RESTful API**：标准的 REST API 设计
- **类型安全**：TypeScript 端到端类型安全
- **错误处理**：统一的错误处理机制
- **速率限制**：API 访问控制和限流

### 5. 安全特性
- **RBAC 权限控制**：细粒度权限管理
- **API 密钥认证**：安全的 API 访问控制
- **会话安全**：安全的会话管理机制
- **环境隔离**：安全的密钥和配置管理

## 部署和扩展性

### 1. 多平台部署
- **Vercel**：一键部署到 Vercel 平台
- **Cloudflare Workers**：边缘计算部署支持

- **自托管**：支持传统服务器部署

### 2. 扩展性设计
- **模块化架构**：支付、AI、存储作为可插拔模块
- **配置驱动**：数据库驱动的功能开关
- **服务层分离**：业务逻辑与表现层分离
- **无服务器优化**：针对无服务器环境的优化

### 3. 开发体验
- **TypeScript**：完整的类型安全支持
- **热重载**：Turbopack 快速开发构建
- **代码质量**：ESLint 和 Prettier 集成
- **自动化测试**：完整的测试框架支持
